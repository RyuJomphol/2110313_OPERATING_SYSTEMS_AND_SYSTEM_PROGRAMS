# Activity 2: Ubuntu Command & C Programming

สภาพแวดล้อมที่ใช้: **Ubuntu 24.04**

เอกสารนี้เป็นเฉลยพร้อมคำอธิบายของคำสั่ง Linux และการเขียนโปรแกรมภาษา C ตามโจทย์ที่กำหนด

---

## 1. แสดงปฏิทินปีที่นิสิตเกิดตลอดทั้งปี

**คำสั่งที่ใช้**

```bash
cal 2005
```

**คำอธิบาย**
คำสั่ง `cal` ใช้สำหรับแสดงปฏิทิน โดยหากระบุปี (เช่น `2005`) ระบบจะแสดงปฏิทินทั้ง 12 เดือนของปีนั้น
ในที่นี้ใช้ `cal 2005` เพราะเป็นปีเกิดของนิสิต จึงสามารถดูปฏิทินตลอดทั้งปีได้ในคำสั่งเดียว

---

## 2. แสดงเฉพาะจำนวนบรรทัดของไฟล์ `/etc/passwd`

**คำสั่งที่ใช้**

```bash
wc -l /etc/passwd
```

**คำอธิบาย**
คำสั่ง `wc` (word count) ใช้สำหรับนับข้อมูลในไฟล์ โดย option `-l` หมายถึงการนับจำนวนบรรทัดเท่านั้น
ไฟล์ `/etc/passwd` เก็บข้อมูลผู้ใช้ในระบบ 1 ผู้ใช้ต่อ 1 บรรทัด ดังนั้นการนับบรรทัดจึงบอกจำนวน user ได้

---

## 3. แสดงเฉพาะจำนวนคำของไฟล์ `/etc/hosts`

**คำสั่งที่ใช้**

```bash
wc -w /etc/hosts
```

**คำอธิบาย**
option `-w` ของ `wc` ใช้สำหรับนับจำนวนคำ (word) ในไฟล์
ไฟล์ `/etc/hosts` เป็นไฟล์ที่เก็บ mapping ระหว่าง IP address และ hostname การนับคำช่วยให้เห็นโครงสร้างข้อมูลภายในไฟล์

---

## 4. แสดง 3 บรรทัดแรกของไฟล์ `/etc/passwd`

**คำสั่งที่ใช้**

```bash
head -3 /etc/passwd
```

**คำอธิบาย**
คำสั่ง `head` ใช้สำหรับแสดงบรรทัดแรกของไฟล์
option `-3` ระบุให้แสดงเฉพาะ 3 บรรทัดแรก เหมาะสำหรับดูตัวอย่างข้อมูลโดยไม่ต้องแสดงทั้งไฟล์

---

## 5. แสดง 4 บรรทัดสุดท้ายของไฟล์ `/etc/hosts`

**คำสั่งที่ใช้**

```bash
tail -4 /etc/hosts
```

**คำอธิบาย**
คำสั่ง `tail` ใช้สำหรับแสดงบรรทัดท้ายของไฟล์
option `-4` หมายถึงแสดง 4 บรรทัดสุดท้าย ซึ่งมักใช้ตรวจสอบข้อมูลที่ถูกเพิ่มเข้ามาล่าสุด

---

## 6. แสดงทุกบรรทัดที่มี username ของนิสิตในไฟล์ `/etc/passwd`

**คำสั่งที่ใช้ (ตัวอย่าง)**

```bash
grep bash /etc/passwd
```

**คำอธิบาย**
คำสั่ง `grep` ใช้ค้นหาบรรทัดที่ตรงกับ pattern ที่กำหนด
ในตัวอย่างใช้คำว่า `bash` เนื่องจาก user ส่วนใหญ่มักใช้ `/bin/bash` เป็น default shell
จึงสามารถใช้ตรวจสอบบรรทัดของ user ที่เกี่ยวข้องได้

> หมายเหตุ: สามารถแทน `bash` ด้วย username จริงของนิสิตได้

---

## 7. โปรแกรมภาษา C แสดงชื่อและนามสกุล พร้อมการ compile และ run

**ไฟล์: `student.c`**

```c
#include <stdio.h>

int main() {
    printf("ชื่อ: ....\n");
    printf("นามสกุล: ....\n");
    return 0;
}
```

**การ compile โปรแกรม**

```bash
gcc student.c -o student
```

**คำอธิบาย**

* `gcc` คือ GNU C Compiler
* `student.c` คือไฟล์ต้นฉบับภาษา C
* `-o student` กำหนดชื่อไฟล์ executable ที่ได้เป็น `student`

**การ run โปรแกรม**

```bash
./student
```

เมื่อรันแล้ว โปรแกรมจะแสดงชื่อและนามสกุลตามที่เขียนไว้ในโค้ด

---

## 8. โปรแกรมภาษา C แสดงรหัสนิสิต และแปลงเป็นภาษา Assembly

**ไฟล์: `student.c`**

```c
#include <stdio.h>

int main() {
    printf("รหัสนิสิต: 65123456\n"); // แก้ไขเป็นรหัสจริงได้
    return 0;
}
```

**การ compile เป็นภาษา Assembly**

```bash
gcc -S student.c -o student.s
```

**คำอธิบาย**

* option `-S` สั่งให้ gcc หยุดที่ขั้นตอนแปลงเป็น Assembly
* `-o student.s` กำหนดชื่อไฟล์ Assembly ที่ได้

**แสดงเนื้อหาไฟล์ Assembly**

```bash
cat student.s
```

คำสั่ง `cat` ใช้แสดงเนื้อหาของไฟล์ทั้งหมดบนหน้าจอ ทำให้เห็นโค้ด Assembly ที่ gcc สร้างขึ้น
