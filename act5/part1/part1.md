# Part 1

โค้ดนี้เป็นโปรแกรม C ที่แสดงการใช้งานเธรด (threads) โดยใช้ไลบรารี pthread เพื่อสร้างและจัดการเธรดหลายตัว

## รายละเอียดโค้ด

- **Header files ที่ใช้:**
  - `<stdio.h>`: สำหรับฟังก์ชัน I/O เช่น printf
  - `<pthread.h>`: สำหรับการจัดการเธรด
  - `<stdlib.h>`: สำหรับฟังก์ชันทั่วไป
  - `<unistd.h>`: สำหรับฟังก์ชัน sleep

- **ฟังก์ชัน `print_arg`:**
  - รับพารามิเตอร์เป็น `void* data`
  - แปลงข้อมูลเป็น `char*` และพิมพ์ออกมาทางหน้าจอด้วย `printf`
  - ใช้ `sleep(20)` เพื่อหยุดทำงานเป็นเวลา 20 วินาที
  - จบเธรดด้วย `pthread_exit(NULL)`

- **ฟังก์ชัน `main`:**
  - ตรวจสอบจำนวนอาร์กิวเมนต์จาก command line ต้องมีอย่างน้อย 3 ตัว (ชื่อโปรแกรม, arg1, arg2)
  - ถ้าไม่เพียงพอ แสดงข้อความ usage และจบโปรแกรม
  - สร้างสองเธรด `t1` และ `t2` โดยเรียก `pthread_create` ส่งฟังก์ชัน `print_arg` และอาร์กิวเมนต์ `argv[1]` และ `argv[2]`
  - ใช้ `pthread_join` เพื่อรอให้ทั้งสองเธรดทำงานเสร็จ
  - จบโปรแกรมด้วย `return 0`

## วัตถุประสงค์
