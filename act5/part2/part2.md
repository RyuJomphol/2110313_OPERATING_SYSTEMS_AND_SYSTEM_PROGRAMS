# Part 2

โค้ดนี้เป็นโปรแกรม C ที่แสดงการสร้างและจัดการเธรดหลายตัว (10 เธรด) โดยใช้ไลบรารี pthread แต่ละเธรดจะพิมพ์หมายเลข ID ของตัวเอง

## รายละเอียดโค้ด

- **Header files ที่ใช้:**
  - `<stdio.h>`: สำหรับฟังก์ชัน I/O เช่น printf
  - `<pthread.h>`: สำหรับการจัดการเธรด
  - `<stdlib.h>`: สำหรับฟังก์ชันทั่วไป

- **ฟังก์ชัน `print_thread_id`:**
  - รับพารามิเตอร์เป็น `void* arg`
  - แปลงเป็น `int*` และ dereference เพื่อได้ค่า ID
  - พิมพ์ข้อความ "This is thread %d" พร้อมหมายเลข ID
  - จบเธรดด้วย `pthread_exit(NULL)`

- **ฟังก์ชัน `main`:**
  - ประกาศอาร์เรย์ `pthread_t threads[10]` สำหรับเก็บ handle ของเธรด
  - ประกาศอาร์เรย์ `int thread_ids[10]` สำหรับเก็บ ID ของแต่ละเธรด
  - ใช้ลูป for เพื่อสร้าง 10 เธรด:
    - กำหนด `thread_ids[i] = i + 1` (ID เริ่มจาก 1 ถึง 10)
    - เรียก `pthread_create` เพื่อสร้างเธรดแต่ละตัว ส่งฟังก์ชัน `print_thread_id` และ address ของ `thread_ids[i]`
  - ใช้ลูป for อีกครั้งเพื่อรอให้ทุกเธรดเสร็จด้วย `pthread_join`
  - จบโปรแกรมด้วย `return 0`

## วัตถุประสงค์
โปรแกรมนี้แสดงให้เห็นการสร้างเธรดหลายตัวที่ทำงานพร้อมกัน แต่ละเธรดจะแสดงหมายเลข ID ของตัวเองเพื่อแสดงความแตกต่าง
